# Emotion-Recognition-System
An intelligent Emotion Recognition System that detects human emotions from facial expressions using Machine Learning and Python. This project helps identify emotional states such as happy, sad, angry, neutral, and more — useful for healthcare, therapy, sentiment analysis, and human–computer interaction.

Features:
1) Real-time face detection
2) Emotion prediction using trained ML model
3) Uses OpenCV for video processing
4) Lightweight and fast inference
5) Supports multiple emotion classes
